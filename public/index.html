URL Shortener

This app takes a URL and returns a shorter version that redirects to the same place.

The original link and new short link are returned in a JavaScript object.

Example Usage:

Passing this: www.example.com/add/http://www.yourURL.com

Returns this: {"url":"http://www.yourURL.com","shortLink":"http://example.com/60397"}
